## Question 4: How can you apply multithreading concepts in real-world applications?

**Your Answer:**

[Multithreading is used in many real applications that we use every day. For example, in web browsers, different parts of a website load at the same time using multiple threads. In mobile apps, one thread is used for the user interface while others handle background tasks like downloading data. In games, threads are used for graphics, sound, and user input at the same time. The Round-Robin scheduling we used in this assignment is similar to how operating systems manage processes fairly. It makes sure no process takes all the CPU time.
